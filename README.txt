Para visualizar los documentos HTML de la página de manera local, siga estas instrucciones:

Instrucciones.


1. Tener XAMPP instalado en el equipo. (si no lo tiene instalado, entrar al siguiente link de la pagina oficial de Apache Friends: https://www.apachefriends.org/es/index.html , descargar y seguir los pasos del instalador)


2. Abrir la carpeta instalada y entrar a htdocs\xampp, en esta dirección se montarán los documentos HTML,CSS,JS(si tiene más de un proyecto crear carpetas separadas para evitar documentos con un mismo nombre). 
Para trabajar en un editor de código, VS Code por ejemplo, se deberá abrir dentro del editor la carpeta xampp.


3. En el caso de trabajar con GitHub, clonar el repositorio en la misma dirección del punto anterior.


4. Abrir el panel de control de XAMPP, iniciar el servidor de Apache y apretar el boton Admin. Para visualizar los documentos HTML, entrar a localhost/xampp, aparecerá el índice de documentos en la carpeta xampp


5. Entrar al documento a visualizar.
